DATASET DOWNLOAD INSTRUCTIONS
-----------------------------
The 'ratings.csv' file is not included in this folder because it is too large (~856 MB).

To get the full dataset:
1. Go to the GroupLens website: https://grouplens.org/datasets/movielens/32m/
2. Download the zip file named 'ml-32m.zip'.
3. Extract the file 'ratings.csv' from that zip and place it in this folder.
